## ğŸ› ï¸ Developer Reference

### Code Organization
- Backend code: `backend/models/`, `backend/routes/`
- Frontend code: `frontend/src/pages/`, `frontend/src/components/`
- Styles: `frontend/src/styles/` (one CSS file per component)

### Key Files to Know
- Backend entry: `backend/server.js`
- Frontend entry: `frontend/src/App.js`
- Frontend router: `frontend/src/App.js`
- Database models: `backend/models/*.js`
- API routes: `backend/routes/*.js`

### Adding New Features
1. Determine if backend API needed
2. Create/modify database model
3. Create/modify API route
4. Create/modify frontend component
5. Add styling
6. Test end-to-end
